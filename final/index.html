Lights Out Game by Prasanna Kumar

Game Instructions

The goal of the game is to turn off all the lights (black tiles) and reveal the image below. Each click toggles the state of the tile clicked and its adjacent tiles (above, below, left, right).

Player:

Target: Turn on all lights 0

Moves: 0

Time: 0:00

How to Play

  • Click on a black tile to toggle it and adjacent tiles.
  • Keep toggling until all tiles are flipped and the image is fully revealed.
  • Check your moves and time to improve your gameplay.

Adendum

This project is about making the Lights Out game. I use HTML, CSS, and JavaScript to make it. First, I make the game with grid and black blocks. User click on block, and it change color or disappear. The target is to make all blocks disappear and show background image.

I also add features like timer, moves counter, and target count. Timer show how much time user take to finish. Moves show how many clicks user did. Target show how many blocks left to win.

For the design, I make page look nice. Background of page is colorful image with balloons. Game box has different image inside it, and blocks cover it like black screen. When user click block, the black block disappear and show image below. This look very nice and fun. I also make hover effect on blocks, so when user move mouse over block, it glow and look better.

The info section has "New Game" button. It also show Target, Moves, and Time in simple design. I style this section with white background, rounded corners, and shadow. It look like card. Button also have hover effect, so it change size and color when mouse go over it.

I face some problem in aligning the grid and showing background image. Sometimes blocks cover the image, and it not look good. I fix it by making blocks transparent when clicked. I also have problem with making page background different from game background, but I solve it using separate CSS.

This project teach me about HTML grid, CSS styles, and JavaScript logic. I learn how to make timer and counter also. It was fun and useful project.
